Pasos del EJERCICIO

Por Mario Cortes

A partir del ejemplo y los archivos entregados, realizar una web que contenga información sobre su trabajo de tesis. Los puntos a considerar son los siguientes:

  1. ingresar a la cuenta git hub y hacer formk a proyecto
  2. abrir consola git y utilizar el comando git clone git:// github.com/chicomario/prueba-git.git
  3. Omitir las imagenes con extensón jpg dentro de la carpeta images. echo "/imagenes/*.jpg" > .gitignore, git commit -n "actualizacion .git ingnore"
  4. Realizar una página por cada elemento del menú indicando los temas importantes de su proyecto de tesis. Se debe realizar un commit al terminar los cambios en cada una.
  5. Crear una rama "diseño" con el propósito de cambiar el aspecto gráfico y guardar los cambios, con al menos 2 commits.
  6. Mezclar los cambios en las dos ramas (Sin eliminar la rama diseño).
  7. Subir los cambios al repositorio y enviarme un correo con el link para acceder.

Importante

Crear un menú llamado "Pasos" que enlace con una página que incluya una descripción y los comandos git utilizados en cada punto.
